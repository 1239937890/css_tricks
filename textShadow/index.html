# 文字颜色渐变

夫天地者万物之逆旅也
光阴者百代之过客也。 
.text-clip{
  color:transparent;
  font-size: 40px;
  font-weight: bold;
  background: linear-gradient(45deg, rgba(0,173,181,1) 0%, rgba(0,173,181,.4)  100%);
  -webkit-background-clip: text;
}

利用的是 -webkit-background-clip: text 属性，也可以将背景颜色设置为背景图

# 文字模糊

而浮生若梦，为欢几何？
.text-blurry{
  text-align: center;
  color: transparent;
  text-shadow: 0 0 5px rgba(0,0,0,0.5);
}
